Example Client Website

This is an example of how the chatbot appears on a client's website.

Note: The chatbot button should appear in the bottom right corner. Click it to start chatting!

How to Add to Your Website

Add this code before the closing </body> tag:

<script 
  defer 
  src="https://YOUR-SERVER-DOMAIN.com/widget/widget.js" 
  data-api-url="https://YOUR-SERVER-DOMAIN.com">
</script>

Replace YOUR-SERVER-DOMAIN.com with your actual chatbot server URL.
